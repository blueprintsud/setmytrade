STEG 1: Välj Analysmodell

Välj den värderingsmodell som passar din investeringsstil bäst

REKOMMENDERAD
Kvantitativ Scoring
Poängbaserad modell med viktade kategorier. Enklast att förstå och använda.
  • 0-100 poäng system
  • 5 viktade kategorier
  • Perfekt för nybörjare
AVANCERAD
DCF - Discounted Cash Flow
Nuvärdesberäkning av framtida kassaflöden. Används av investment banks.
  • Framtida kassaflöden
  • Diskonteringsränta
  • För erfarna investerare
UTDELNING
Dividend Discount Model
Värdering baserad på framtida utdelningar. Perfekt för inkomstinvesterare.
  • Utdelningsfokus
  • Dividend growth rate
  • För stabila bolag
TILLVÄXT
PEG Ratio Model
P/E justerat för tillväxt. Identifierar undervärderade tillväxtaktier.
  • P/E / Growth Rate
  • Värdering vs tillväxt
  • För growth stocks
VALUE
Graham Number / Value Investing
Benjamin Grahams formel med säkerhetsmarginal. För värdeinvesterare.
  • Säkerhetsmarginal
  • Konservativ värdering
  • För value investors
